About

The GroundedWren Text Adventure Suite (GTAS) is lightweight, accessible software for creating and playing robust single-player text-based games.

Built from scratch using only fundamental web technologies, GTAS was designed from the ground-up for screen readers first, not as an afterthought. These are text games, after all.

GTAS is the passion project of just one caffeine-addicted woman. If you have feedback, ideas, or questions, please reach out to vera@groundedwren.com

The code for this project is freely available; if it will help you, please use it for good.
